The goal of this website is to make the activities of Edmonton City Council more accessible!

To get notified when new meeting information is released you can add this RSS feed to any RSS reader.

You can search for terms you are interested in the Search tab. Searching will also create a custom RSS feed for only those terms that you can use!

This website parses City Council meeting minutes from https://pub-edmonton.escribemeetings.com.

This project is open source. You can view the code at https://github.com/SamuelmdLow/EdmontonCityCouncilFeed

Contact samuel.m.d.low@gmail.com with any concerns or suggestions!
